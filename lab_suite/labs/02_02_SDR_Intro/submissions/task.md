## Task



Mit diesen 3 Notebooks wird erstmals der **RTL-SDR** Software Defined Radio USB Stick verwendet, dabei werden Samples von einem **authentischen Funksignal** eingelesen, in unserem Fall sind es **UKW FM Radiosender**.

Öffne das erste Notebook und navigiere durch die Notebooks, es wird gezeigt, wie man den USB-Treiber aktiviert, das RTL-SDR device öffnet und samples einliest, die als Zeitsignal und Spektrum dargestellt werden. Dabei wird **matplotlib** zur grafischen Darstellung in steigender Komplexität verwendet, bis hin zu **interaktiven Slider- und Button Widgets**.

Es gibt keine expliziten Aufgaben zu lösen, versuche, den Python code zu verstehen, experimentiere mit PArameter und den Slider-Widgets.
