# 05 – Arrays (NumPy)

## Lernziele (Gerüst)

- **Kein C-Array in Python:** Python hat keinen eingebauten Typ „Array“ wie in C; Listen sind flexibel, aber für numerische Rechnungen (Signale, Spektren) nutzt man **NumPy-Arrays**.
- **NumPy:** `numpy.array`, Erzeugen aus Listen, Index/Slice wie bei Listen, elementweise Operationen.
- **Bezug zu KT:** Vorbereitung auf **02.Signale und Spektren** – diskrete Signale, Abtastwerte, einfache Beispiele aus dem Kontext Signale/Spektren.
