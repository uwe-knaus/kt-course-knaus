# 02 – Datentypen

## Lernziele (Gerüst)

- **Zahlen:** `int` (ganze Zahlen), `float` (Dezimalzahlen); einfache Literale.
- **Strings:** `str`, Anführungszeichen, mehrzeilige Strings (optional).
- **Boolesche Werte:** `True`, `False` (z. B. für spätere Bedingungen).
- **Typen prüfen:** `type(...)` zur Veranschaulichung.
