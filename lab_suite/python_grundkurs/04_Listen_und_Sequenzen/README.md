# 04 – Listen und Sequenzen

## Lernziele (Gerüst)

- **Listen:** `list`, Erstellen mit `[]`, Elemente durch Komma.
- **Index:** Zugriff mit `[0]`, `[1]`, negative Indizes (`[-1]`).
- **Slice:** `[start:end]`, `[start:end:step]` (kurz).
- **Einfache for-Schleife:** `for x in liste:` – über Elemente iterieren.

## Material (geplant)

- Beispiele: Liste von Zahlen, von Strings; Schleife mit `print`.
- Optional: Länge `len(liste)`.

## Anknüpfung KT

- Grundlage für spätere Signale/Arrays (02.Signale Spektren) und für Codierung (Symbolfolgen als Listen). Noch keine direkte KT-Anwendung nötig.
