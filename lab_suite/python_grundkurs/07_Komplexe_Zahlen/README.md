# 07 – Komplexe Zahlen

## Lernziele (Gerüst)

- **Bezug zu KT:** Vorlesungskapitel **02.Signale und Spektren** – komplexe Zahlen für Zeiger, Phasoren, Spektren, Bandpass-Signale; Grundlage für FFT und Spektraldarstellung.
- **Programmierung vs. Mathematik:** In Python sind komplexe Zahlen ein **eingebauter Datentyp** (`z = 3+4j`); Zugriff auf Real-/Imaginärteil, Betrag, Phase; Unterschied zu C (z. B. struct oder zwei Variablen).
- **Darstellung mit Matplotlib:** Zeitdiagramm (Real- und Imaginärteil als zwei Kurven in einem Plot), XY-Diagramm (komplexe Zahlenebene: Realteil = x, Imaginärteil = y).
