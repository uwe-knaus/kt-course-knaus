# 03 – Variablen und Operatoren

## Lernziele (Gerüst)

- **Zuweisung:** Name = Wert; Variablen als Speicher für Ergebnisse.
- **Arithmetik:** `+`, `-`, `*`, `/`, `//`, `%`, `**`.
- **Vergleichsoperatoren:** `==`, `!=`, `<`, `>`, `<=`, `>=` (Ergebnis: `bool`).
- **Ausdrücke:** Kombination aus Variablen, Literalen und Operatoren.

## Material (geplant)

- Beispiele: Berechnungen in Variablen speichern, Ergebnisse ausgeben.
- Vorbereitung für Formeln (z. B. Entropie: Logarithmus → siehe 07_Module für `math`).
