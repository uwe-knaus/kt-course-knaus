# 14 – Dateien und Ein-/Ausgabe

## Lernziele (Gerüst)

- **Dateien:** `open(...)`, Modi `"r"` / `"w"` / `"a"`, `read()` / `readline()` / `write()`, `close()` oder `with open(...) as f:`.
- **Konsole:** `input()` zum Einlesen, `print()` (bereits bekannt) zur Ausgabe.
- **Kurz:** Zeichenketten formatieren (f-Strings, `.format()`).

## Material (geplant)

- Beispiele: Textdatei lesen (z. B. eine Zeile pro Symbol), einfache Auswertung; Ergebnis in Datei schreiben oder ausgeben.

## Anknüpfung KT

- Optional: Auswertung von Messdaten, Logs, oder Symbolfolgen aus Dateien in späteren KT-Übungen. Kann auch weggelassen werden, wenn der Fokus auf Rechnen und Plots liegt.
