# 00 – Einführung

## Lernziele (Gerüst)

- **Was ist Python?** Kurz: interpretierte, vielseitige Programmiersprache; Einsatz in Wissenschaft, Web, Automatisierung.
- **Wann entwickelt?** Entstehung (Guido van Rossum, frühe 1990er), Meilensteine (Python 2/3).
- **Warum heißt es Python?** Angelehnt an Monty Python (humorvolle Referenz), nicht an die Schlange.
- **Konzept Interpreter:** Programm wird Zeile für Zeile (bzw. Bytecode) ausgeführt, kein separater Kompilerschritt nötig – kurze Erklärung ohne tiefe Implementierungsdetails.
- **Konzept Objektorientierung:** Alles in Python ist ein Objekt (Zahlen, Listen, Funktionen); Klassen als Bauplan. Nur als Idee, noch kein Code.
