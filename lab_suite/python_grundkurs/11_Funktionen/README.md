# 11 – Funktionen

## Lernziele (Gerüst)

- **Definition:** `def name(parameter):` mit Einrückung des Rumpfes.
- **Parameter und Rückgabe:** Argumente übergeben, `return` Wert.
- **Aufruf:** `ergebnis = name(argumente)`.

## Material (geplant)

- Beispiele: einfache Funktion (z. B. Entropie H(p) oder mittlerer Binärstellenaufwand als Funktion mit Parametern).
- Optional: Mehrere Parameter, Default-Werte.

## Anknüpfung KT

- **01.2 Informationstheorie:** Entropie oder mittlerer Binärstellenaufwand als wiederverwendbare Funktion mit Parametern (M, p, …).
