# 15 – Besondere Datentypen

## Lernziele (Gerüst)

- **`None`:** Bedeutung „kein Wert\", typische Verwendung (z. B. optionale Rückgabe, Platzhalter).
- **`bytes` und `bytearray`:** Unveränderliche bzw. veränderliche Bytefolgen; Zusammenhang mit Binärdateien und Codierung.
- **`str.encode()` und `bytes.decode()`:** Umwandlung Zeichenkette ↔ Bytes (z. B. UTF-8, ASCII); Relevanz für Übertragung und Dateien.
- **Optional:** `set` (Menge, eindeutige Elemente ohne Reihenfolge) – kurzer Überblick.

## Material

- Notebook: None, bytes/bytearray, encode/decode, einfache Beispiele; optional set.
